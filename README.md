# HireWise AI

An AI-powered career assistant that helps students improve their resumes and prepare for interviews.

## Features

- JWT Authentication
- Google Login
- Resume Upload
- Resume Parsing
- ATS Score
- AI Resume Analysis
- Dashboard
- Job Matching (Coming Soon)
- AI Interview (Coming Soon)

## Tech Stack

Frontend
- Angular
- Tailwind CSS

Backend
- Node.js
- Express

Database
- MongoDB Atlas

Authentication
- JWT
- Google OAuth

AI
- Gemini API
